# Magmos Docs

Magmos is a composable yield-dollar protocol on Sui with:

- AURUM minting and redemptions
- sAURUM yield accumulation
- Live allocation routing across Scallop, Aftermath, and DeepBook
- Operator-side safety rails and thermal limits

## What is in this docs set

- `quickstart`: run and test locally
- `protocol`: core protocol concepts
- `operator`: infra and monitoring surface
